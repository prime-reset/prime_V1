# PRIME V1 — Trading Discipline OS

Prototype front-end React / Next.js de l'application PRIME.

## Lancer le projet

```bash
npm install
npm run dev
```

Puis ouvrir :

```bash
http://localhost:3000
```

## Ce qui est inclus

- Dashboard PRIME
- Profil trader
- Checklist dynamique
- Score discipline
- Détection d'erreurs comportementales
- Prescription intelligente
- Mode reset
- Journal / timeline
- Design mobile-first noir/blanc premium
